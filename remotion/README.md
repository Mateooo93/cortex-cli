# cortex-cli promo video

Remotion project for the cortex-cli marketing video.

## Preview

```bash
npm run dev
```

Opens Remotion Studio at `CortexPromo` (1920×1080, 18s).

## Render

```bash
npm run render
```

Writes `../assets/cortex-cli-promo.mp4`.

Requires Chromium (bundled by Remotion on first render).
